## 포트번호
cmd 창에서 아래 코드를 입력하면 현재 활성화되어있는 포트 번호가 나옵니다.
```
netstat -ano
```


포트 번호를 삭제하려면 pid 이용하면 됩니다.
```
taskkill /f /pid 0000
```


## 아이피
아이피를 확인하려면 cmd 창에 아래 코드를 입력하면 됩니다.
```
ipconfig
```

`IPv4 주소`라는 것이 있는데 localhost 대신 사용 가능합니다.
